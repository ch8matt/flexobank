# flexobank


## Clone the project:

`git clone https://github.com/ch8matt/flexobank/`


## Go to the project dir:

`cd flexobank`

## Use a virtual environement:

```
mkdir venv
python3 -m venv venv
source venv/bin/activate
```

## Install the mandatory dependencies:

`pip install -r requirements.txt`

## To launch:

`python main.py`
